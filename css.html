Depurando CSS e Seletores

[Voltar]

Basicamente depurar css é a ferramenta inspecionar no navegador, nos possibilitando ver todo nosso código.

Já os seletores definem quais elementos queremos aplicar nas nossas propriedades CSS. Eu também posso modificar a cor do corpo todo o body tanto quanto elementos específicos como tags de blocos e listas, por exemplo:

  • Eu gosto de pastel e os meus favoritos são de queijo e pizza.
  • Eu gosto de gatos, tenho um cinza, dois pretos, três tipos frajola e um rajado.
  • Eu gosto de ghost, mas a minha música favorita é shinunoga E-Wa.
  • Eu uso instagram, no entanto minha rede favorita é o tik tok.

Podemos usar o seletor por ID para identificar uma parte do texto que queremos destacar, como por exemplo no primeiro paragráfo do texto, um parágrafo de boas vindas, identificando com a tag id dentro da tag p, podemos destaca-lo no style, para que ele salte aos olhos do leitor no backgroud e cor de texto escolhido.



Existem 5 tipos de seletores, seletor de tags,seletor de id,seletor de classes,seletores universais e seletores de atributo.

  • Seletor de tags é um seletor básico que escolhe todos os elementos que correspondem com o nome fornecido.
  • Seletor de id funciona com o atributo id só podendo existir apenas um elemento com o mesmo ID no mesmo documento.
  • Seletor de classes é outro seletor básico, que escolhe elementos baseados no valor de seu atributo .
  • Seletores universal. Este seletor básico irá escolher todos os nós. Ele também existe em um namespace único e em uma variante de todo o namespace também.
  • Seletores de atributo. Este seletor básico ira escolher baseado no valor de um dos atributos, ou até mesmo pelo próprio atributo.

Margin coloca/aplica uma margem nos elementos, com o margin e top, left e right, além do inherit que copia o estilo do elemento acima.

A propriedade padding define uma a distância entre o conteúdo de um elemento e suas bordas. É um atalho que evita definir uma distância para cada lado separadamente (padding-top, padding-right, padding-bottom, padding-left).
